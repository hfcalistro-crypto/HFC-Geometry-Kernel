# Roadmap

## Fase 1 - Fundacao

- Estrutura modular do projeto.
- Documentacao inicial.
- Testes de estrutura.
- Definicao dos principios de engenharia.

## Fase 2 - Nucleo Tecnico

- Modelos basicos de geometria.
- Leitura inicial de malhas.
- Contratos internos entre modulos.

## Fase 3 - Reverse CAD

- Deteccao de primitivas geometricas.
- Reconstrucao parametrica assistida.
- Validacao matematica dos resultados.

## Fase 4 - IA e Interface

- Sugestoes assistidas por IA.
- Visualizacao 3D.
- Fluxos simples para usuarios nao especialistas.

## Fase 5 - Exportacao e Plugins

- Exportacao para formatos CAD.
- Gerenciamento de plugins.
- Integracoes externas controladas.
