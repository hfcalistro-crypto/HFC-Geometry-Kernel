# HFC Reverse CAD

**Engenharia Reversa para Todos**

HFC Reverse CAD e um software proprietario em Python para conversao de malhas 3D em CAD parametrico com apoio de IA.

O objetivo do projeto e oferecer uma base profissional, modular e extensivel para fluxos de engenharia reversa, mantendo a experiencia simples para o usuario e a validacao tecnica sob responsabilidade de motores matematicos confiaveis.

## Status

Estrutura inicial do projeto. Funcionalidades avancadas ainda nao foram implementadas.

## Propriedade

Este software e proprietario. Todos os direitos sao reservados ao proprietario do projeto HFC Reverse CAD.
